# Runtime Federation Workspace Rule

## Runtime Workspace

Grafana acts as Runtime Federation Workspace.

* Federation
* KPI
* Runtime synchronization
* Queue propagation
* ODD constraint
* Node synchronization

* AUTHORIZED / HOLD / STOP visualization
